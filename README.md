\# 🛒 Sistema PDV Multi-loja



Sistema de Ponto de Venda (PDV) desenvolvido em Python com Streamlit, voltado para controle de vendas, gestão de pagamentos e relatórios em ambiente multi-loja.



\## 🚀 Acesse o sistema

👉 https://pdv-camargo.onrender.com/



\## 📌 Funcionalidades



\- Registro de vendas

\- Controle de pagamentos (Dinheiro, PIX, Cartão e Misto)

\- Histórico de vendas

\- Relatórios por período

\- Controle de múltiplas lojas



\## 🛠 Tecnologias utilizadas



\- Python

\- Streamlit

\- SQL

\- Pandas



\## 💡 Sobre o projeto



Este projeto foi desenvolvido com o objetivo de aplicar na prática conceitos de desenvolvimento de sistemas, banco de dados e lógica de programação, com foco na área de tecnologia.
